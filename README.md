## Structure du projet
- `Tableau.h / Tableau.cpp` : classe de base pour gérer un tableau de valeurs  
- `TableauBorne.h / TableauBorne.cpp` : classe dérivée pour gérer un tableau d’éléments bornés entre une valeur minimale et maximale  
- `main.cpp` : programme principal pour tester les fonctionnalités des classes  

## Fonctionnalités
- Création d’intervalles avec bornes minimum et maximum  
- Affichage des valeurs contenues dans un intervalle  
- Vérification de la validité des intervalles  
- Héritage et utilisation de classes de base pour gérer des tableaux d’intervalles
